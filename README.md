# React App Boilerplate

## Try

### Install packages

```bash
yarn
```

### Run GraphQL Server

How to: https://github.com/lzl/graphql-server-boilerplate

### Go dev!

```bash
npm run dev
```

### Go prod!

```bash
npm run build
npm run start
```

### Tips

The `package.json` file will be your best friend.
